\section*{Problem(6.4)}
If P(A/B)$>$P(A), then which of the following is correct :\\
(A) P(B/A)$<$P(B)\\
(B) P(AB)$<$P(A).P(B)\\
(C) P(B/A)$>$P(B)\\
(D) P(B/A)=P(B)
\section*{Solution(6.4)}

Given P(A/B)$>$P(A)\\
and since P(A/B) = $\frac{P(AB)}{P(B)}$ we have
\begin{align}
    \tag{6.4.1}
    \frac{P(AB)}{P(B)}>P(A)\\
    \tag{6.4.2}
    \implies P(AB)>P(A).P(B)
\end{align}
Hence, option (B) is false. Now, dividing the equation by P(A) on both sides i.e.,
\begin{align}
    \tag{6.4.3}
    \frac{P(AB)}{P(A)}>P(B)   
\end{align}
But $\frac{P(AB)}{P(A)}$ = P(B/A). Therefore, from (6.4.3)
\begin{align}
    \tag{6.4.4}
    P(B/A)>P(B)
\end{align}
Hence, option (C) is correct.
